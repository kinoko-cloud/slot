# 表示ルール

## モード別表示

### 閉店後モード（22:30〜翌10:00）
- **目的**: 前日結果の振り返り + 翌日の準備
- **表示するもの**:
  - 前日の爆発台TOP10（差枚・ART・RB・確率・連チャン・最大枚数・グラフ）
  - 翌日のおすすめ台（根拠 + 直近3日間データ）
  - 答え合わせ（予測 vs 実績）
- **表示しないもの**:
  - 空き / 遊技中 / 営業外（意味がない）
  - リアルタイムデータ（存在しない）
  - 天井情報（次の営業日まで無意味）

### 営業中モード（10:00〜22:30）
- **目的**: リアルタイムで狙い台を判断
- **表示するもの**:
  - 空き / 遊技中
  - 本日のART・RB・確率・G数
  - 天井到達回数・現在のハマりG数
  - 設定推定（G数が十分な台）
- **加えて表示**:
  - 根拠（なぜおすすめか）
  - 直近3日間データ（判断材料）

## 全ページ共通ルール

### 台カード（v2統一フォーマット）
1. **根拠文章**（一番上）
2. **直近3日間データ**（前日・前々日・3日前）
   - 各日: ART / RB / 確率 / 差枚 / 最大連チャン
   - データなし → 「データなし」と明示
   - 全データなし → 「過去データなし（新台 or 台移動の可能性）」
3. **設定推定**（G数3000以上で表示）
4. 展開時: 全根拠 + 蓄積傾向

### 台番号
- 4桁ゼロパディング（1→0001, 23→0023, 0752→0752）

### 曜日傾向（店舗単位）
- 台個別ではなく、店舗全体の傾向
- 「店舗名＋曜日＋具体的な根拠」で表示
  - 例: 「アイランド秋葉原は水曜が高設定投入日（過去実績: 好調率80%）」
  - 悪い例: 「水曜★3」（★の定義が不明）

### 蓄積傾向
- 「N日連続不調 → 設定上げの可能性あり」
- 「N日連続好調 → 据え置き期待」（2日）
- 「N日連続好調 → 据え置き or 転落警戒」（3日以上）
